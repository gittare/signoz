# Elasticsearch

Monitor your Elasticsearch cluster using SigNoz to gain comprehensive insights into its health, performance, and resource utilization. This integration collects key metrics across nodes, indices, clusters, and caches.

## Track Key Elasticsearch Metrics

Monitoring Elasticsearch is crucial for maintaining a healthy and performant cluster. Key metrics provided by this integration include:

*   **Node Metrics:** Monitor disk usage, CPU utilization, and JVM memory to ensure each node has sufficient resources.
*   **Cluster Health:** Track the overall health status, active and unassigned shards, and pending tasks to maintain cluster stability.
*   **Index Metrics:** Monitor document counts, disk usage per index, and search query performance to optimize indexing and search operations.
*   **Query and Request Metrics:** Keep an eye on search request rates and query latencies to identify performance bottlenecks.
*   **Cache Metrics:** Track cache evictions, hits, and misses to optimize cache utilization.
